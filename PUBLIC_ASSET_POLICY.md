# Public Asset Policy

This repository is public and must not contain copyrighted game assets copied from the original arcade release, console ports, ROM dumps, MAME extractions, soundtrack rips, or derivative sprite/audio sheets.

## Allowed
- Original code written for this remake
- Replacement artwork created for this project
- Replacement sound effects and music created for this project
- Design notes, metadata, and gameplay research expressed in text
- Test fixtures that do not embed copyrighted source assets

## Not Allowed
- Original Raiden II sprites, backgrounds, portraits, UI, audio, or FMV
- ROM-derived spritesheets, tilemaps, or sound banks
- Decompiled game data copied into repo files
- Temporary “just for now” copyrighted assets committed to any branch

## Working Rule
If reference material is needed during development, keep it **outside** the tracked repository and never commit it.
